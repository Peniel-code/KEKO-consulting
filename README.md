# KEKO Consulting - Site Web Officiel

Cabinet d'expertise comptable et de conseil au Togo, fondé en 2009 par KEKE Yaovi Fernand.

## 🚀 Fonctionnalités

- **Site multi-pages responsive** : Accueil, À propos, Services, Contact, Blog
- **Optimisation SEO complète** : Meta tags, structured data, sitemap, robots.txt
- **Accessibilité** : WCAG 2.1 AA compliant, navigation clavier, screen readers
- **Performance** : Core Web Vitals optimisés, lazy loading, PWA ready
- **Sécurité** : Headers sécurisés, validation des formulaires

## 📊 Statistiques SEO

- **Pages indexées** : 5 pages principales
- **Mots-clés cibles** : comptabilité Togo, audit entreprise, conseil fiscal
- **Structured Data** : AccountingService, Organization, LocalBusiness
- **Performance** : Score Lighthouse > 90

## 🛠 Technologies Utilisées

- **HTML5** sémantique avec microdonnées
- **CSS3** avec Grid, Flexbox et animations
- **JavaScript** vanilla avec Intersection Observer
- **PWA** features avec manifest et service worker
- **SEO** optimisé pour Google, Bing

## 📁 Structure du Projet

```
├── index.html          # Page d'accueil

```
